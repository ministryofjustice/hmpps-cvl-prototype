Create a new proto version

We will use v18 as the last version and we are creating v19 as an example.
In protos:

  1. app > routes > v18.js
    • Duplicate latest version (for example v18)
    • Rename to next version (example v19)
    • Open v19.js
    • On line 3: var version = '/v18'; change to var version = '/v19';

  2. app > routes.js
    • Copy and paste whole line require('./routes/v18.js')(router);
    • Change to new version require('./routes/v19.js')(router);

  3. app > views
    • Duplicate latest version (v18) folder
    • Rename v19
    • Open v19
    • Select folder
    • Search 'in folder' for v18 and replace with v19

  4. app > views > v19 > additional
    • Select folder
    • Search 'in folder' for v18 and replace with v19

  5. app > views > index
    • Copy the last version text:
    • Change
      • v18to >v19
    • On old v18 version change In Design to ready for UR (can use the html form block from v17)
